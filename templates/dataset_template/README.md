## Running the Pipeline

See [CRISPR Pipeline Documentation](../../docs/CRISPR_PIPELINE.md) for detailed instructions.

### Quick Start

```bash
# 1. Configure scripts (edit BASE_DIR and ACCESSION)
vim bin/1_CRISPR_pipeline/1_generate_per_sample_metadata.sh

# 2. Generate sample metadata
./bin/1_CRISPR_pipeline/1_generate_per_sample_metadata.sh

# 3. Upload to GCP (dry run first)
DRY_RUN=true ./bin/1_CRISPR_pipeline/2_upload_to_gcp.sh
./bin/1_CRISPR_pipeline/2_upload_to_gcp.sh

# 4. Run pipeline
./bin/1_CRISPR_pipeline/3_run_CRISPR_pipeline.sh
```
